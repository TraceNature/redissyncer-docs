#### 1.3根据任务状态查询任务

[comment]: <> (<img src="img/image-20210805110235565.png" alt="image-20210805110235565" style="zoom:50%;" />)

任务状态共有8种

* creating        任务创建中
* created         任务创建完成
* stop            任务停止
* starting        任务启动中
* finish          任务已完成
* broken          任务异常
* rdbrunning      任务全量数据同步中
* commandrunning  任务增量数据同步中
